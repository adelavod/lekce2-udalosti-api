Vlastnosti události

Každá událost má event object - sadu informací o události.

  • onkeydown dáváme do < body >
  • u console.log(udalost) se vypíše celá sada informací o události
  • u console.log(udalost.key) se vypíše jen konkrétní klávesa, která byla stisknuta
  • u každé události je event object složený z jiných informací
